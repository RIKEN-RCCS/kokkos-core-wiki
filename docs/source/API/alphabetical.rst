アルファベット順のAPI
=====================

ここに列挙されているすべての関数とクラスは、 ``Kokkos::`` 名前空間の一部です。

アルゴリズム
------------

+--------------------------+--------------+------------------+------------------------------------------------------------+
| 名前                     | アルゴリズム | カテゴリー       | 説明                                                       |

+--------------------------+--------------+------------------+------------------------------------------------------------+

| init                     | アルゴリズム | ランダムナンバー | Random_XorShift64_Poolのシードを使用して状態を初期化       |
